# LFDA Files Organizer

Aplicación Java para distribuir archivos en carpetas.

## Características

- Dispone de interfaz de usuario (GUI)
- Permite copiar los archivos contenidos en la carpeta origen en carpetas de destino con un máximo de archivos por carpeta.

 ## Instrucciones

 - Seleccionar carpeta de origen (Source folder).
 - Seleccionar carpeta de destino (Target folder).
 - Indicar el prefijo que tendrá cada carpeta creada dentro de la carpeta de destino.
 - Indicar el número de ficheros por carpeta (Files per folder).
 - Pulsar el botón "Organize"

## Proceso
- Se crearán tantas carpetas como sean necesarias con el prefijo indicado y una numeración secuencial.
- En cada carpeta se copiarán el número de ficheros indicado.
- Los ficheros no se eliminan de la carpeta original.
